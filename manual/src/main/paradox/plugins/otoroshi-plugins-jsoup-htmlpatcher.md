
# Html Patcher

## Infos

* plugin type: `transformer`
* configuration root: `HtmlPatcher`

## Description

This plugin can inject elements in html pages (in the body or in the head) returned by the service



## Default configuration

```json
{
  "HtmlPatcher" : {
    "appendHead" : [ ],
    "appendBody" : [ ]
  }
}
```
