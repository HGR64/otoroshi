
# Public quotas

## Infos

* plugin type: `validator`
* configuration root: `ServiceQuotas`

## Description

This plugin will enforce public quotas on the current service







## Default configuration

```json
{
  "ServiceQuotas" : {
    "throttlingQuota" : 100,
    "dailyQuota" : 10000000,
    "monthlyQuota" : 10000000
  }
}
```
